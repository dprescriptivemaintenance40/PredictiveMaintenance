Example of uni-variate parameter:

Here TD1 represents only one "primary variable". In real case, there are multiple primary and derived parameters.

For simplicity, we have demonstrated “linear regression” modeling on uni variate, with following characteristics-

1) TD1 in present context is Compressor 1st stage discharge pressure.
2) SCADA transmitted sensor data @1 hr interval has been collected over past 3+ years.
3) Initial data cleansing removes input collected during standstill state. Next layer cleansing allocates upper & lower limits considering seasonality.
4) Each machine is unique, and hence the diagnostics model varies, as guided by DPM specialists, and workshop with plant engineers.

This example is part of real case study conducted at one of the renowned O&G companies. Data are real. And the example clearly explains the intricacies of developing a solution for any process machinery.
